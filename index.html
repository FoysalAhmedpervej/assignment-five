Important FAQ

What is the document Object Model (DOM) ?

The Document Object Model (DOM) is the data representation of the objects that comprise the structure and content of a document on the web.t represents the structure of a webpage as a hierarchical tree that can be accessed and manipulated using programming languages, most commonly JavaScript.

How do you select an element from the DOM ?

I can select an element using methods like document.getElementById('id') to get an element by ID or document.querySelector('.class') for the first element matching a selector. For multiple elements, Iâ€™d use document.getElementsByClassName('class'). These methods help me access elements for further manipulation in my code.

What is event delegation in the context of DOM, and why is it useful ?

Event delegation is a technique where I attach a single event listener to a parent element, allowing it to handle events on its child elements by leveraging event bubbling. This reduces the number of listeners and makes it easier to handle dynamically added elements, improving both performance and maintainability.

How do you manipulate an element's attributes and styles using the DOM ?

I manipulate an element's attributes with methods like element.setAttribute('attr', 'value') or element.getAttribute('attr') to retrieve values. For styles, I use element.style.propertyName = 'value', such as element.style.color = 'red' to change an element's text color directly via JavaScript.
